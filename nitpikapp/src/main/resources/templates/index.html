NikPik is a photo-management web application equipped with the necessary features users need for securely managing and sharing photos.
Basic features include allowing users to easily manage (upload, tag, or delete) their media. For organization and categorization, users can create customizable
tags and tag their media for easier searchability. Such features make NitPik perfect for those who are nitpicky about how they want to organize their photos.
